Fracada requires only blockfrost credentials as environment variables.
These are defined in env.js:
```
const BLOCKFROST_API_URL = process.env.NEXT_PUBLIC_BLOCKFROST_API_URL || '';
const BLOCKFROST_PROJECT_ID = process.env.NEXT_PUBLIC_BLOCKFROST_PROJECT_ID || '';
const NETWORK_NAME = process.env.NEXT_PUBLIC_NETWORK_NAME || 'Preprod';
```
